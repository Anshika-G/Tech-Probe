INSTRUCTIONS

  • There will be 15 MCQ based questions divided in 3 levels.

  • For the first 6 questions, we have a timer. Timer will be dependant on the question. Once the timer is over, user will be out of the game.

  • Credit points will be awarded to the user for each level. Credit points will be level dependant.

  • To help user correctly and timely answer the questions, we are providing lifelines. (They will help stimulate the mind of the user in the correct direction)

  • Lifelines which will be provided to the user: Double Dip (in which user can make 2 guesses), Flip The Question (in which at user's request, one question with another one is replaced) and the last one in which some hint will be given which will help the user to decode the question.

  • After giving 7 correct answers, if the user hasn't used any of the lifelines he'll be awarded extra credit points.

  • Team with the highest credit points will win, in case of a tie there'll be a tie breaker bonus question (like faster finger first). Team that'll answer the question first will be declared as the winners.
